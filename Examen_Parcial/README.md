Informe Final - Examen Parcial
Con la realización de esta práctica, se logró integrar con éxito una base de datos relacional en MySQL con un programa desarrollado en C++. 
El sistema ahora es capaz de gestionar la información de manera centralizada, permitiendo que los datos de alumnos, cursos y secciones 
se almacenen de forma permanente.

Puntos clave logrados:
Integridad de la información: Se implementó la lógica para reconocer cuando un alumno ya está asignado a un curso, evitando registros duplicados
según los requerimientos.

Consultas Relacionales: Mediante el uso de JOIN, el programa genera listados legibles donde se vincula el nombre del alumno con su curso y 
sección correspondiente.

Estructura Modular: El código se organizó separando las funciones de conexión y consulta para facilitar el flujo eficiente de los datos.

En conclusión, el proyecto cumple con los requisitos técnicos y las validaciones de base de datos solicitadas para este examen parcial.
